# gh-deployment-workflow

Este é um projeto de exemplo para demonstrar a implantação contínua de um site estático no GitHub Pages usando GitHub Actions.

## Como funciona

Qualquer alteração no arquivo `index.html` na branch `main` irá acionar um workflow do GitHub Actions que construirá e publicará o site automaticamente.

## Acesso ao Site

O site será publicado em `https://nilo-lima.github.io/gh-deployment-workflow/`.
